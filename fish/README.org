* manual installation

  #+begin_example
git clone https://github.com/fish-shell/fish-shell
mkdir build; cd build
CC=/path/to/bin/gcc CXX=/path/to/bin/g++ \
cmake -DCMAKE_INSTALL_PREFIX=~/opt/fish/bin ..
make
make install
  #+end_example  

If gcc is old, you should specify a newer gcc to cmake.
